# Security Policy

OpenBridge takes security and the safety of users' funds very seriously.

## Reporting a Vulnerability

If you discover a security vulnerability, please report it **privately**. Do not open a public issue.

**Preferred way to report:**
- Send an email to: **Saghi8499@gmail.com**

### What to include in your report:
- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Any suggested fix (if available)

We will acknowledge your report within **48 hours** and aim to resolve critical issues as quickly as possible.

---

Thank you for helping keep OpenBridge secure.
